— FAQ

Common questions, answered.

What FiveM framework do these scripts target? +
QBox is the primary target. QBCore is supported as a fallback via auto-detection at startup. The scripts use ox_lib + oxmysql universally and ox_inventory where item handling is needed. No script requires our other scripts to run — they're all standalone, with optional integration if you own multiple.
Will my server need code changes to install these? +
No. Each script is a self-contained resource. Drop into your resources/ folder, add ensure lines to server.cfg, import the SQL, restart. Configuration is done via the config.lua in each resource — no code changes needed.
Is the source obfuscated? +
No. All source is plain Lua, well-commented, and public on GitHub. Your developers can audit it, extend it, and modify it for your server's specific needs.
How does the integration between scripts work? +
Every script exposes FiveM exports that other resources can call. Reputation exposes AddReputation, GetTier, etc. Events optionally grants criminal rep on wins. DOJ optionally uses reputation tier to modify bail and sentencing. Businesses optionally applies legal-rep-tier discounts to customers. Each integration is wrapped in pcall so missing dependencies never crash anything.
Can I try a script before paying? +
The source is already public on GitHub — read it, audit it, understand exactly what you're buying before purchase. For serious server owners we also offer evaluation copies on request via Discord or email.
What's your license / can I resell? +
Single-server use. Each license covers one production FiveM server. Reselling, redistributing, or modifying for resale is not permitted. Lifetime updates are included for original purchasers — no monthly fees.
How do you handle bug reports and support? +
GitHub issues on the public repos for tracking. Discord for live discussion. Bug fixes are typically shipped the same day they're reported, especially for paying customers.
Are you offering custom development? +
For serious servers — yes. Contact us with what you're building and we'll discuss scope, timeline, and rates.
